# Roast & Bloom — Artisanal Coffee (Vue 3 + Vite + Pinia + Tailwind)

A modern, responsive e‑commerce storefront for a fictional artisanal coffee brand.

## Stack
- Vue 3 (Composition API)
- Vite
- Pinia (cart store)
- Vue Router
- Tailwind CSS

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
